## Installation

### 1. Trouvez Olostep dans Relay.app

Recherchez "Olostep" dans Relay.app lors de la création d'un nouveau flux de travail :

1. Allez sur [Olostep sur Relay.app](https://www.relay.app/apps/olostep/integrations)
2. Cliquez sur "Commencer gratuitement" ou "Créer un flux de travail"
3. Recherchez "Olostep" dans le sélecteur d'applications
4. Sélectionnez l'application Olostep

Alternativement, visitez directement la [page d'intégration Olostep](https://www.relay.app/apps/olostep/integrations) pour parcourir les actions disponibles et créer votre premier flux de travail.

### 2. Connectez Votre Compte

Lorsque vous utilisez Olostep pour la première fois dans un flux de travail, vous serez invité à connecter votre compte :

1. Cliquez sur "Connecter Olostep"
2. Entrez votre clé API Olostep
3. Autorisez la connexion

Obtenez votre clé API depuis le [Tableau de bord Olostep](https://olostep.com/dashboard).

## Actions Disponibles

### Scraper URL

Extraire le contenu d'une seule URL. Prend en charge plusieurs formats et le rendu JavaScript.

**Cas d'utilisation :**
- Surveiller des pages spécifiques pour des changements
- Extraire des informations produits de sites e-commerce
- Rassembler des données d'articles de presse ou de blogs
- Extraire du contenu pour l'agrégation de contenu

**Configuration :**

<ParamField path="URL" type="string" required>
  URL du site web à scraper (doit inclure http:// ou https://)
</ParamField>

<ParamField path="Format" type="dropdown" default="Markdown">
  Choisissez le format : Markdown, HTML, JSON, ou Texte Brut
</ParamField>

<ParamField path="Country Code" type="string">
  Code pays pour le contenu spécifique à une localisation (par ex., "US", "GB", "CA")
</ParamField>

<ParamField path="Wait Before Scraping" type="integer">
  Temps d'attente en millisecondes pour le rendu JavaScript (0-10000)
</ParamField>

<ParamField path="Parser" type="string">
  ID de parseur optionnel pour une extraction spécialisée (par ex., "@olostep/amazon-product")
</ParamField>

**Champs de sortie :**
- ID de Scraping
- URL Scrappée
- Contenu Markdown
- Contenu HTML
- Contenu JSON
- Contenu Texte
- Statut
- Horodatage
- URL de capture d'écran (si disponible)
- Métadonnées de la page

**Exemples de Flux de Travail :**

<AccordionGroup>
  <Accordion title="Surveiller les Prix des Concurrents">
    **Déclencheur :** Planification (Tous les jours à 9h)
    
    **Action :** Olostep - Scraper URL
    - URL : Page produit du concurrent
    - Format : JSON
    - Parseur : @olostep/amazon-product
    
    **Action :** Google Sheets - Créer une Ligne
    - Ajouter les données de prix au tableau de suivi
    
    **Boucle Humaine :** Examiner les changements de prix
    - Approuver les baisses de prix significatives
    
    **Action :** Slack - Envoyer un Message
    - Alerter l'équipe des changements de prix approuvés
  </Accordion>

  <Accordion title="Extraire et Sauvegarder des Articles de Blog">
    **Déclencheur :** Flux RSS - Nouvel Élément
    
    **Action :** Olostep - Scraper URL
    - URL : \{\{URL de l'article\}\}
    - Format : Markdown
    
    **Action :** Notion - Créer une Page
    - Sauvegarder le contenu de l'article dans la base de données Notion
  </Accordion>

  <Accordion title="Enrichissement de Leads avec l'IA">
    **Déclencheur :** Google Sheets - Nouvelle Ligne
    
    **Action :** Olostep - Scraper URL
    - URL : Site web de l'entreprise depuis la feuille
    - Format : Markdown
    
    **Action IA :** OpenAI - Extraire des Données
    - Extraire : taille de l'entreprise, industrie, produits
    
    **Action :** Google Sheets - Mettre à Jour la Ligne
    - Ajouter les données enrichies à la feuille
  </Accordion>
</AccordionGroup>

## Avantages de Relay.app

Relay.app offre plusieurs avantages par rapport aux plateformes d'automatisation traditionnelles :

### Flux de Travail avec Boucle Humaine

Ajoutez des étapes de révision manuelle directement dans vos flux de travail :

```
1. Olostep scrappe le site web
2. Étape de Révision Humaine ← Réviser et approuver les données
3. Sauvegarder dans la base de données uniquement si approuvé
```

Parfait pour :
- Le contrôle qualité avant le traitement des données
- Le jugement humain sur le contenu extrait
- Les flux de travail d'approbation pour les données sensibles
- La révision des résumés générés par l'IA

### Fonctionnalités d'IA Intégrées

Utilisez l'IA directement dans vos flux de travail sans intégrations externes :

- **Prise de Décision IA** : Laissez l'IA décider des chemins de flux de travail basés sur le contenu extrait
- **Filtrage Intelligent** : Filtrer et catégoriser automatiquement les données extraites
- **Résumé de Contenu** : Résumer de longs articles avant de les stocker
- **Extraction de Données** : Utilisez l'IA pour extraire des données structurées à partir de contenu non structuré

### Meilleure UX et Tests

L'interface conviviale de Relay.app facilite la création de flux de travail :

- Informations de test claires
- Messages d'erreur utiles
- Constructeur de flux de travail visuel
- Débogage et dépannage faciles

## Exemples de Flux de Travail Populaires

### Surveillance des Prix en E-commerce avec Révision

Surveillez les prix des concurrents avec approbation humaine :

```
Déclencheur : Planification (Toutes les heures)
↓
Action : Olostep - Scraper URL
  - URL : Page produit du concurrent
  - Format : JSON
  - Parseur : @olostep/amazon-product
↓
Étape de Révision Humaine
  - Réviser le changement de prix
  - Approuver ou rejeter
↓
Action : Slack - Envoyer un Message (Si approuvé)
  - Alerte : "Le prix a changé à $\{\{prix\}\}"
```

### Agrégation de Contenu avec l'IA

Agrégerez et améliorez le contenu de plusieurs sources :

```
Déclencheur : Google Sheets - Nouvelle Ligne
↓
Action : Olostep - Scraper URL
  - URL : \{\{URL depuis la feuille\}\}
  - Format : Markdown
↓
Action IA : Résumer le Contenu
  - Générer un résumé
↓
Action : Airtable - Créer un Enregistrement
  - Stocker l'article avec résumé IA
```

### Pipeline d'Enrichissement de Leads

Enrichissez les données de leads avec des informations web et l'IA :

```
Déclencheur : HubSpot - Nouveau Contact
↓
Action : Olostep - Scraper URL
  - URL : \{\{Site web de l'entreprise\}\}
  - Format : Markdown
↓
Action IA : Extraire Infos Entreprise
  - Extraire : taille de l'entreprise, industrie, produits
↓
Étape de Révision Humaine
  - Vérifier l'exactitude de l'extraction IA
↓
Action : HubSpot - Mettre à Jour le Contact
  - Ajouter les données enrichies vérifiées
```

## Dépannage

<AccordionGroup>
  <Accordion title="Échec de l'Authentification">
    **Erreur** : "Clé API invalide"
    
    **Solutions** :
    - Vérifiez la clé API depuis le [tableau de bord](https://olostep.com/dashboard)
    - Assurez-vous qu'il n'y a pas d'espaces supplémentaires dans la clé API
    - Reconnectez votre compte Olostep dans Relay.app
    - Vérifiez que la clé API est active
  </Accordion>

  <Accordion title="Le Scraping Renvoie du Contenu Vide">
    **Erreur** : Les champs de contenu sont vides
    
    **Solutions** :
    - Augmentez le temps "Attendre avant de scraper"
    - Vérifiez si le site web nécessite une connexion
    - Essayez un format différent (HTML vs Markdown)
    - Vérifiez que l'URL est accessible
    - Vérifiez si le site bloque l'accès automatisé
  </Accordion>

  <Accordion title="Limite de Taux Dépassée">
    **Erreur** : "Limite de taux dépassée"
    
    **Solutions** :
    - Espacer les exécutions de flux de travail avec des délais
    - Utiliser le traitement par lots au lieu de scrapes individuels
    - Mettez à niveau votre plan Olostep
    - Vérifiez la limite de taux dans le tableau de bord
  </Accordion>

  <Accordion title="URL Non Scrappée">
    **Erreur** : Des URL spécifiques échouent à scraper
    
    **Solutions** :
    - Vérifiez le format de l'URL (inclure http:// ou https://)
    - Vérifiez si l'URL nécessite une authentification
    - Testez l'URL dans un navigateur d'abord
    - Essayez avec le paramètre de pays
    - Contactez le support pour les domaines bloqués
  </Accordion>
</AccordionGroup>

## Relay.app vs Zapier

Relay.app est une alternative moderne à Zapier avec plusieurs avantages :

**Avantages de Relay.app :**
- **Boucle Humaine** : Étapes de révision manuelle intégrées
- **IA Intégrée** : Fonctionnalités d'IA intégrées aux flux de travail
- **Meilleure UX** : Interface plus intuitive et messages d'erreur plus clairs
- **Meilleur Prix** : Plus rentable pour de nombreux cas d'utilisation
- **Intégrations Plus Profondes** : Intégrations plus complètes avec des applications populaires

**Quand Choisir Zapier :**
- Besoin d'intégration avec des applications moins couramment utilisées
- Préférence pour une plateforme plus établie
- Nécessité de fonctionnalités spécifiques à Zapier

Pour des comparaisons détaillées, voir [la comparaison de Relay.app avec Zapier](https://www.relay.app/blog/zapier-alternatives).

## Tarification

Olostep facture en fonction de l'utilisation de l'API, indépendamment de Relay.app :

- **Scrapes** : Paiement par scrape
- **Lots** : Paiement par URL dans le lot
- **Crawls** : Paiement par page crawlée
- **Maps** : Paiement par opération de carte

Consultez la tarification actuelle sur [olostep.com/pricing](https://olostep.com/pricing).

**Plan Relay.app** : Vous avez également besoin d'un plan actif Relay.app pour exécuter des flux de travail.
