Par défaut, Olostep prend en charge le retour de contenu dans divers formats, y compris :
- HTML
- Texte
- PDF brut
- Markdown

**Pour certains cas d'utilisation, tu pourrais ne pas avoir besoin de l'intégralité du contenu, mais seulement de données spécifiées, structurées et propres au format JSON.**

Pour y parvenir, tu peux utiliser : 

- **[Parsers](./parsers)** : Idéal pour le scraping de sites web à fort volume, cohérent et récurrent
- **[Extraction LLM](./llm-extraction)** : Idéal pour des besoins d'extraction flexibles ou des sites web avec des structures changeantes.

Pour obtenir l'accès à la fonctionnalité d'Extraction LLM, merci de nous contacter à info@olostep.com

Les deux méthodes fournissent des données JSON propres et structurées qui peuvent être immédiatement utilisées dans tes applications sans traitement supplémentaire.
