Avec les contextes, tu peux transmettre des cookies personnalisés à l'API. Cela te permet de réutiliser des cookies, l'authentification et les données stockées pour alimenter les automatisations IA.

### Créer un Contexte

Voici comment tu peux créer un contexte pour transmettre des cookies personnalisés à l'API :

1. Installe le plugin de navigateur Olostep depuis ici : [Plugin de navigateur Olostep](https://chromewebstore.google.com/detail/olostep/mjalbmdidmklbojfgakdpbifkgjjilkh)
2. Visite le site web pour lequel tu souhaites enregistrer les cookies/l'authentification
3. Connecte-toi au site web avec les identifiants que tu veux enregistrer
4. Clique sur l'icône du plugin Olostep et sélectionne "Soumettre le Contexte"
5. Copie le context_id (qui sera quelque chose comme `ctx_yylqbfayto`) depuis le plugin et utilise-le dans tes requêtes API

### Utiliser le Contexte dans les requêtes API

```python
import requests
import json

url = "https://api.olostep.com/v1/scrapes"

payload = {
    "url_to_scrape": "https://www.feizy.com/account-dashboard",
    "formats": ["html", "markdown"],
    "remove_css_selectors": "none",
    "country": "US",
    "context": {
        "id": "ctx_yylqbfayto"
    }
}

headers = {
    "Authorization": "Bearer <API_KEY>",
    "Content-Type": "application/json"
}

response = requests.request("POST", url, json=payload, headers=headers)

# Analyser la réponse JSON
data = response.json()

# Afficher le JSON avec une indentation correcte
print(json.dumps(data, indent=4))
```
