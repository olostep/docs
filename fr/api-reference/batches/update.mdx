---
title: "Mettre à jour un lot"
description: "Met à jour les métadonnées pour un lot spécifique. Seules les métadonnées peuvent être mises à jour après la création du lot."
openapi: "fr/openapi/batches.json PATCH /v1/batches/{batch_id}"
---

<Tip>
**Sémantique de fusion :** Les mises à jour des métadonnées suivent l'approche de Stripe — les nouvelles clés sont ajoutées, les clés existantes sont mises à jour, et les clés définies sur une chaîne vide `""` sont supprimées.
</Tip>
