## EXPLANATION
WITH OLOSTEP, EVERY SCRAPE HAPPENS WITH JS RENDERING ENABLED BY DEFAULT. THIS AVOIDS ANTI-BOT MEASURES AND IS ESPECIALLY USEFUL FOR WEBSITES THAT USE FRAMEWORKS LIKE REACT, ANGULAR, VUE, ETC.
IT ALSO ALLOWS YOU TO SCRAPE WEBSITES THAT USE JS TO LOAD THEIR CONTENT, SUCH AS INFINITE SCROLLING, LAZY LOADING, ETC.
