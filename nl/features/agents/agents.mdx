## Maak een agent

Maak een agent met een natuurlijke taalprompt en een doelmodel.

<CodeGroup>

```python Python
API_URL = 'https://api.olostep.com/v1/agents' # endpoint beschikbaar voor geselecteerde klanten
API_KEY = '<YOUR_API_KEY>'

headers = {
  'Authorization': f'Bearer {API_KEY}', 
  'Content-Type': 'application/json'
}

data = {
  "prompt": """
    Vind portfoliobedrijven van https://www.vcsheet.com/funds en 
    schrijf naar een Google Sheet met kolommen (Fondsnaam, Fonds Website URL, 
    Fonds LinkedIn URL, Portfolio Bedrijfsnaam, Portfolio Bedrijf URL, 
    Portfolio Bedrijf LinkedIn URL). Voer wekelijks uit op maandag om 9:00 uur 
    en e-mail naar steve@example.com wanneer er nieuwe bedrijven worden toegevoegd.
  """,
  "model": "gpt-4.1"
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()
print(result)
```

```js Node
const res = await fetch('https://api.olostep.com/v1/agents', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer <YOUR_API_KEY>', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'Vind portfoliobedrijven van https://www.vcsheet.com/funds en schrijf naar een Google Sheet...',
    model: 'gpt-4.1'
  })
})
console.log(await res.json())
```

```bash cURL
curl -s -X POST "https://api.olostep.com/v1/agents" \
  -H "Authorization: Bearer $OLOSTEP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Vind portfoliobedrijven van https://www.vcsheet.com/funds en schrijf naar een Google Sheet...",
    "model": "gpt-4.1"
  }'
```

</CodeGroup>

## Mogelijkheden

De Olostep agent kan:

- **Webonderzoek**: Automatisch zoeken en gegevens van websites extraheren
- **Gegevensorganisatie**: Informatie structureren in spreadsheets, databases of andere formaten
- **Geplande Uitvoering**: Taken uitvoeren volgens een terugkerend schema (dagelijks, wekelijks, op een vooraf bepaalde tijd)
- **Meerstapsworkflows**: Complexe, meerledige onderzoekstaken autonoom uitvoeren
- **Meldingen**: E-mailwaarschuwingen sturen wanneer nieuwe gegevens worden gevonden of taken zijn voltooid
- **Aangepaste Uitvoer**: Gegevens retourneren als CSV, JSON, Google Sheet, of direct in je database
