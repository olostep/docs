---
title: "Batch Bijwerken"
description: "Werkt de metadata voor een specifieke batch bij. Alleen metadata kan worden bijgewerkt na het aanmaken van de batch."
openapi: "nl/openapi/batches.json PATCH /v1/batches/{batch_id}"
---

<Tip>
**Merge-semantic:** Metadata-updates volgen de aanpak van Stripe — nieuwe sleutels worden toegevoegd, bestaande sleutels worden bijgewerkt, en sleutels die zijn ingesteld op een lege string `""` worden verwijderd.
</Tip>
