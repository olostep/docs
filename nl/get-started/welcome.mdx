Olostep biedt AI een manier om het web te doorzoeken, gestructureerde data in real-time te extraheren en aangepaste onderzoeksagenten te bouwen.

## Introductie

- De **Olostep API** is de beste **webzoek**, **scraping** en **crawling** API voor AI, gebruikt door enkele van de toonaangevende startups ter wereld.

- De **Olostep Agent** maakt het mogelijk om **onderzoeksworkflows** op een no-code manier te automatiseren met slechts een prompt in natuurlijke taal.


## Overzicht van endpoints

Dit zijn de kernendpoints van Olostep:

  <Card title="/scrapes →" href="/features/scrapes/scrapes">

  <Card title="/batches →" href="/features/batches/batches">

  </Card>

  <Card title="/answers →" href="/features/answers/answers">

  </Card>
