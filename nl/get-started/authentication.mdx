De API-eindpunten vereisen dat je je authenticeert met een API-token.

## Genereer een token

Het token kan worden gegenereerd vanuit het Olostep-dashboard. Maak een account aan [hier](https://www.olostep.com/auth/).

## Gebruik je token

Je kunt je authenticeren door een `Authorization` header toe te voegen aan al je HTTP-aanroepen. De Authorization header is als volgt geformatteerd: `Authorization: Bearer <API-TOKEN>` (vervang `<API-TOKEN>` door je token. Als je geen token hebt, kun je er gratis een genereren vanuit het Olostep [dashboard](https://www.olostep.com/dashboard/).

Voorbeelden:

<CodeGroup>

```python Python
# pip install requests

import requests

endpoint = 'https://api.olostep.com/v1/scrapes/<SCRAPE_ID>'
headers = {
    'Authorization': 'Bearer <API-TOKEN>',
    'Accept': 'application/json'
}

response = requests.get(endpoint, headers=headers)
print(response.status_code)
print(response.json())
```

```js Node
// npm install node-fetch

// ESM
import fetch from 'node-fetch'

const endpoint = 'https://api.olostep.com/v1/scrapes/<SCRAPE_ID>'
const res = await fetch(endpoint, {
  headers: {
    'Authorization': 'Bearer <API-TOKEN>',
    'Accept': 'application/json'
  }
})
console.log(res.status)
console.log(await res.json())
```

```bash cURL
curl -L -X GET 'https://api.olostep.com/v1/scrapes/<SCRAPE_ID>' \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer <API-TOKEN>'
```
