De geverifieerde [Olostep Web Scraper-node](https://n8n.io/integrations/olostep-web-scraper/) biedt je zes operaties binnen de visuele builder van n8n: een URL scrapen, het web doorzoeken, AI-antwoorden krijgen, duizenden URL's batch-scrapen, een site crawlen of al zijn links in kaart brengen.

[Bekijk op n8n →](https://n8n.io/integrations/olostep-web-scraper/)

## Voordat je begint

- **Een Olostep-account met een API-sleutel:** [krijg er gratis een](https://olostep.com/dashboard), geen creditcard vereist. Je eerste 500 credits zijn inbegrepen.
- **n8n draait:** ofwel [n8n Cloud](https://n8n.io/cloud/) of een zelf-gehoste instantie. Community-nodes moeten ingeschakeld zijn (ze zijn standaard ingeschakeld op de meeste installaties).
- **Geen codering vereist:** alles in deze gids wordt gedaan via de visuele editor van n8n.

---

## Installatie

<Steps>
  <Step title="Zoek naar de Olostep-node">
    Open een workflow, klik op **+**, en zoek naar **Olostep**. Selecteer **Olostep Web Scraper** uit de resultaten.

    ![Zoek naar Olostep in de n8n-nodekiezer](/images/integrations/n8n/step-1.png)
  </Step>

  <Step title="Installeer de node">
    Klik op het resultaat om het Node-details paneel te openen en klik vervolgens op **Node installeren**. n8n zal `n8n-nodes-olostep` installeren en je vragen om opnieuw op te starten. Doe dat voordat je verder gaat.

    ![Olostep Web Scraper node details met Install node knop](/images/integrations/n8n/step-2.png)

    <Note>
      Als **Community Nodes** is uitgeschakeld voor je werkruimte, moet een beheerder het eerst inschakelen. Zie de [n8n community nodes gids](https://docs.n8n.io/integrations/community-nodes/installation/).
    </Note>
  </Step>

  <Step title="Voeg je API-sleutel toe">
  Open de Olostep-node in je workflow, klik op **Credential instellen** (in het Parameters-tabblad), voeg je API-sleutel toe en klik op **Opslaan**.

    ![Olostep credentials formulier in n8n met API Key veld](/images/integrations/n8n/step-3.png)

    Haal je sleutel op van het [Olostep dashboard →](https://olostep.com/dashboard)
  </Step>

  <Step title="Verbind en voer uit">
    Verbind de Olostep-node met een trigger en eventuele downstream-stappen, en voer vervolgens je workflow uit.

    ![n8n workflow canvas met Schedule Trigger verbonden met Olostep-node](/images/integrations/n8n/step-4.png)
  </Step>
</Steps>

## Acties

<CardGroup cols={2}>
  <Card title="Website Scrapen" icon="file-lines">
    Haal inhoud van elke URL als Markdown, HTML, JSON of platte tekst. Behandelt JS-gerenderde pagina's met optionele wachttijden en landgerichte targeting.
  </Card>

  <Card title="Zoeken" icon="magnifying-glass">
    Voer een webzoekopdracht uit en krijg gestructureerde resultaten (titels, URL's en fragmenten) als JSON.
  </Card>

  <Card title="Antwoorden (AI)" icon="sparkles">
    Stel een vraag in natuurlijke taal en krijg een antwoord met geciteerde bronnen. Handig voor LLM-nodes wanneer je onderbouwde antwoorden nodig hebt.
  </Card>

  <Card title="Batch Scrape URL's" icon="layer-group">
    Dien tot 10.000 URL's in één taak in, verwerkt in parallel. Geeft een `batch_id` terug; haal resultaten asynchroon op.
  </Card>

  <Card title="Crawl Maken" icon="spider-web">
    Begin vanaf een URL, volg links en scrape alle subpagina's. Goed voor documentatiesites, blogs of volledige site-inname. Geeft een `crawl_id` terug.
  </Card>

  <Card title="Kaart Maken" icon="map">
    Verkrijg elke URL op een site zonder inhoud te scrapen. Gebruik het voor ontdekking vóór een batchtaak. Geeft een `map_id` terug.
  </Card>
</CardGroup>

<Note>
  **Batch, Crawl, en Map zijn asynchroon.** Bewaar de geretourneerde ID en gebruik een Wacht-node of een tweede workflow om resultaten op te halen zodra de verwerking is voltooid.
</Note>

## Voorbeeldworkflow: Lead verrijking vanuit Google Sheets

**Wat het doet:** Wanneer je een bedrijfs-URL in een Google Sheet plakt, scrapt deze workflow automatisch de website van het bedrijf, extraheert belangrijke informatie met een AI-node en schrijft de resultaten terug naar dezelfde rij, waardoor een lege spreadsheet wordt omgezet in een ingevulde lead-database.

**Gebruikte nodes:** Google Sheets trigger → Olostep Scrape Website → OpenAI → Code → Google Sheets update

![Lead verrijking workflow in n8n: Google Sheets trigger verbonden met Olostep, OpenAI, Code, en Google Sheets update nodes](/images/integrations/n8n/workflow-example.png)

---

### Stap 1: Stel je Google Sheet in

Maak een sheet met deze kolommen: `Company URL`, `Industry`, `Description`, `Company Size`, `Enriched`. De workflow leest van `Company URL` en vult de rest in.

### Stap 2: Voeg een Google Sheets trigger toe

Voeg in n8n een **Google Sheets** trigger-node toe. Stel de gebeurtenis in op **Row Added**, wijs het naar je sheet en stel het in om de `Company URL` kolom te bekijken. Nu, elke keer dat je een nieuwe URL in de sheet plakt, wordt deze workflow geactiveerd.

### Stap 3: Voeg Olostep Scrape Website toe

Verbind een **Olostep Web Scraper** node na de trigger. Stel in:
- **Actie:** Website Scrapen
- **URL:** `{{ $json["Company URL"] }}` (haalt de URL uit de nieuwe rij)
- **Uitvoerformaat:** Markdown

Markdown werkt hier het beste omdat het navigatie, advertenties en boilerplate verwijdert. De AI-node in de volgende stap krijgt schone proza over het bedrijf in plaats van ruwe HTML-ruis.

### Stap 4: Voeg een OpenAI-node toe

Verbind een **OpenAI** node. Stel het model in op `gpt-4o-mini` (snel en goedkoop voor extractietaken) en gebruik deze prompt:

```
Je bent een verkooponderzoeker. Op basis van de onderstaande inhoud van de bedrijfswebsite, extraheer:
1. Industrie (één zin, bijv. "B2B SaaS", "E-commerce", "Gezondheidszorg")
2. Eén-zin bedrijfsbeschrijving (max 20 woorden)
3. Geschatte bedrijfsgrootte (Startup / SMB / Mid-market / Enterprise)

Geef alleen een JSON-object terug met sleutels: industry, description, company_size.

Website-inhoud:
{{ $json.markdownContent }}
```

Het `markdownContent` veld is wat Olostep retourneert van de scrape, als schone platte tekst.

### Stap 5: Parseer de AI-reactie en schrijf terug

Voeg een **Code** node toe om de JSON van OpenAI te parseren:

```js
const parsed = JSON.parse($input.first().json.message.content);
return [{ json: parsed }];
```

Verbind vervolgens een **Google Sheets** node ingesteld op **Update Row**. Map de kolommen:
- `Industry` → `{{ $json.industry }}`
- `Description` → `{{ $json.description }}`
- `Company Size` → `{{ $json.company_size }}`
- `Enriched` → `Yes`

### Wat je krijgt

Plak een URL zoals `https://notion.so` in je sheet, en binnen ~10 seconden vult de rij zich in:

| Company URL | Industry | Description | Company Size | Enriched |
|---|---|---|---|---|
| https://notion.so | Productivity SaaS | All-in-one werkruimte voor notities, documenten en databases | Mid-market | Yes |

Vanaf hier kun je deze workflow uitbreiden: voeg een Slack-melding toe wanneer verrijking is voltooid, filter op industrie voordat je terugschrijft, of vervang Google Sheets door HubSpot om contacten direct bij te werken.

## Parsers

Voeg een parser-ID toe aan het **Parser** veld bij elke Scrape of Batch-actie om gestructureerde data te krijgen in plaats van ruwe inhoud:

| Parser | Extraheert |
|--------|------------|
| `@olostep/amazon-product` | Titel, prijs, beoordeling, recensies, afbeeldingen, varianten |
| `@olostep/google-search` | Resultaten titels, URL's, fragmenten |
| `@olostep/google-maps` | Bedrijfsnaam, adres, beoordeling, recensies |
| `@olostep/extract-emails` | E-mailadressen van elke pagina |
| `@olostep/extract-socials` | Sociale profiel links (X, GitHub, LinkedIn, etc.) |
| `@olostep/extract-calendars` | Google Calendar en ICS-links |

Bekijk de volledige lijst in de [Olostep parser store →](https://www.olostep.com/store)

## Probleemoplossing

<AccordionGroup>
  <Accordion title="API-sleutel geweigerd">
    Kopieer de sleutel direct van [olostep.com/dashboard](https://olostep.com/dashboard) zonder spaties aan het einde. Verwijder en maak de credential opnieuw aan in n8n als de fout aanhoudt.
  </Accordion>

  <Accordion title="Gescrapete inhoud is leeg">
    Verhoog **Wacht Voor Scrapen** (probeer 2000–5000ms voor JS-zware pagina's). Bevestig dat de URL openbaar toegankelijk is zonder inlog. Als een specifiek domein consequent faalt, neem contact op met [info@olostep.com](mailto:info@olostep.com).
  </Accordion>

  <Accordion title="Batch URL formaatfout">
    Het **URL's om te Scrapen** veld verwacht een JSON-array:
    ```json
    [
      { "url": "https://example.com/page-1", "custom_id": "p1" },
      { "url": "https://example.com/page-2", "custom_id": "p2" }
    ]
    ```
    Gebruik een Code-node stroomopwaarts om deze array uit je data te bouwen indien nodig.
  </Accordion>

  <Accordion title="Rate limit bereikt">
    Voeg een **Wacht** node toe tussen scrape-stappen, of schakel over naar **Batch Scrape URL's** in plaats van enkele scrapes te herhalen. Controleer het huidige gebruik in het [dashboard](https://olostep.com/dashboard).
  </Accordion>

  <Accordion title="Community Nodes niet zichtbaar in Instellingen">
    Op n8n Cloud moeten community-nodes worden ingeschakeld door een werkruimte-eigenaar. Op zelf-gehoste, zorg ervoor dat `N8N_COMMUNITY_PACKAGES_ENABLED=true` is ingesteld in je omgeving. Zie [n8n's installatiegids](https://docs.n8n.io/integrations/community-nodes/installation/).
  </Accordion>
</AccordionGroup>
