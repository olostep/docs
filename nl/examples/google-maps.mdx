## Overzicht

De API van Olostep stelt je in staat om gestructureerde gegevens van Google Maps locaties te extraheren door gebruik te maken van parsers. Deze parsers transformeren de ruwe HTML van Google Maps pagina's naar schone, gestructureerde JSON-gegevens die klaar zijn voor analyse of integratie in je applicaties.

## Integratie Voorbeeld

Om geparseerde JSON-inhoud van Google Maps resultaten te verkrijgen, moet je `json` opnemen in de formats parameter en de naam van de parser `@olostep/google-maps` specificeren in het parser object. Hier is hoe je Google Maps locatiegegevens in een gestructureerd formaat kunt ophalen:

```python
import requests
import json

url = "https://api.olostep.com/v1/scrapes"
payload = {
"url_to_scrape": "https://www.google.com/maps/search/Coffee+on+the+Rocks/@40.3699261,-105.5321621,17z?hl=en",
"formats": ["json"],
"parser": {"id": "@olostep/google-maps"},
"remove_css_selectors": "none",
"wait_before_scraping": 4000
}
headers = {
"Authorization": "Bearer <YOUR-API-KEY>",
"Content-Type": "application/json"
}
response = requests.request("POST", url, json=payload, headers=headers)
print(json.dumps(response.json(), indent=4))
```

- **Basisverzoekinformatie**: `id`, `object`, `created` timestamp, `url_to_scrape`
- **Resultaat object** met URL's om toegang te krijgen tot verschillende formaten van de gegevens
- **json_content** met gestructureerde zoekresultaten inclusief:

- `Bedrijfsgegevens` (naam, adres, contactinformatie)
- `Beoordelings- en recensie-informatie`
- `Openingstijden`
- `Gebruikersrecensies met tekst en afbeeldingen`
- `Vergelijkbare plaatsen in de buurt`
- `Populaire tags en voorzieningen`

Marktonderzoek: Analyseer klantrecensies en -sentimenten over bedrijven in specifieke gebieden
