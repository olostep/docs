## Overzicht

De API van Olostep stelt je in staat om gestructureerde data uit zoekmachineresultaten te halen door gebruik te maken van [parsers](../../features/structured-content/parsers). Deze parsers transformeren de ruwe HTML van zoekresultaten naar schone, gestructureerde JSON-data die klaar is voor analyse of integratie in je applicaties.

## Integratievoorbeeld

Om geparseerde JSON-inhoud van zoekresultaten te krijgen, moet je `json` opnemen in de `formats` parameter en de naam van de parser `@olostep/google-search` specificeren in het `parser` object.

Hier is hoe je Google zoekresultaten in een gestructureerd formaat kunt ophalen:

```python
import requests
import json

endpoint = "https://api.olostep.com/v1/scrapes"
payload = {
    "formats": ["json"],
    "parser": {"id": "@olostep/google-search"},
    "url_to_scrape": "https://www.google.com/search?q=alexander+the+great&gl=us&hl=en",
    "wait_before_scraping": 0,
}
headers = {
    "Authorization": "Bearer <YOUR-API-KEY>",
    "Content-Type": "application/json"
}
response = requests.request("POST", endpoint, json=payload, headers=headers)
print(json.dumps(response.json(), indent=4))
```

De respons bevat:

- **Basisinformatie over het verzoek**: `id`, `object`, `created` timestamp, `url_to_scrape`
- **Resultaatobject** met URL's om toegang te krijgen tot verschillende formaten van de data
- **json_content** met gestructureerde zoekresultaten inclusief:
- `searchParameters`: Informatie over de zoekopdracht
- `knowledgeGraph`: Gedetailleerde informatie over het zoekonderwerp (indien beschikbaar)
- `organic`: Lijst van zoekresultaten met titel, link, positie en snippet
- `peopleAlsoAsk`: Gerelateerde vragen die gebruikers vaak zoeken
- `relatedSearches`: Voorgestelde gerelateerde zoekopdrachten

`json_content` is het belangrijkste deel van de respons met de gestructureerde zoekresultaten. Je kunt de JSON-inhoud direct uit de respons halen of de gehoste URL gebruiken die in de respons wordt gegeven.

## Belangrijke Opmerkingen

**Zoekparameters**: De `gl=us` en `hl=en` parameters stellen de geolocatie in op de VS en de taal op Engels. Pas deze aan indien nodig.

## Conclusie
Het doorgeven van aangepaste parsers aan de API stelt je in staat om alleen de gestructureerde data te krijgen die je wilt. Dit maakt het eenvoudig om zoekresultaten in je applicaties te integreren, data-analyse uit te voeren, of zoekgerelateerde functies te bouwen.

Voor informatie over beschikbare parsers of om een aangepaste parser aan te vragen voor jouw specifieke gebruikssituatie, neem contact met ons op via [info@olostep.com](mailto:info@olostep.com).
