---
title: "Retrieve Batch"
description: "Get information about a specific batch"
api: "GET https://batches.olostep.com/batches/{batch_id}"
---

Returns detailed information about a specific batch, including its current status and progress.

## Base URL

```bash
https://batches.olostep.com
```

## Authentication

Bearer token authentication is required:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash
curl https://batches.olostep.com/batches/batch_xyz123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

```json
{
  "batch_id": "batch_xyz123",
  "status": "in_progress",
  "total_urls": 100,
  "completed_urls": 45,
  "number_retried": 0,
  "start_epoch": 1698757200,
  "start_date": "2024-10-31",
  "completion_percentage": "45.00"
}
```

## Fields

| Field                   | Type   | Description                                  |
|-------------------------|--------|----------------------------------------------|
| batch_id                | String | Unique identifier for the batch             |
| status                  | String | Current status: "in_progress", "completed", or "failed" |
| total_urls              | Number | Total number of URLs in the batch           |
| completed_urls          | Number | Number of successfully processed URLs       |
| number_retried          | Number | Number of retry attempts made               |
| start_epoch             | Number | Batch creation timestamp in Unix epoch      |
| start_date              | String | Batch creation date (YYYY-MM-DD)            |
| completion_percentage   | String | Percentage of URLs successfully processed  |

## Error Codes

| Status | Description                           |
|--------|---------------------------------------|
| 401    | Invalid or missing authentication token |
| 404    | Batch not found                       |
| 500    | Internal server error                 |
