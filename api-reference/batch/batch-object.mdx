## Overview

The batch object contains all information about a batch processing request, including its status, progress, and timestamps.

## Object Structure

```json
{
  "id": "batch_xyz123",
  "object": "batch",
  "status": "in_progress",
  "created": 1698757200000,
  "total_urls": 100,
  "completed_urls": 45,
  "number_retried": 0,
  "batch_parser": "none",
  "country": "IT",
  "start_date": "2024-10-31"
}
```

## Fields

| **Field**           | **Type**  | **Description**                                   |
|---------------------|-----------|--------------------------------------------------|
| **id**              | `String`  | Unique identifier for the batch                  |
| **object**          | `String`  | Type identifier, always `"batch"`                |
| **status**          | `String`  | Current status: `"in_progress"` or `"completed"` |
| **created**         | `Number`  | Batch creation timestamp in milliseconds         |
| **total_urls**      | `Number`  | Total number of URLs in the batch                |
| **completed_urls**  | `Number`  | Number of successfully processed URLs             |
| **number_retried**  | `Number`  | Number of retry attempts made                     |
| **batch_parser**    | `String`  | Parser used for the batch                        |
| **country**   | `String`  | Country selected for node allocation              |
| **start_date**      | `String`  | Batch creation date (`YYYY-MM-DD`)               |

## Status Values

| **Status**      | **Description**                               |
|-----------------|-----------------------------------------------|
| `in_progress`   | Batch is currently being processed           |
| `completed`     | All URLs have been processed successfully    |
