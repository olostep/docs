Olostep offre all'AI un modo per cercare sul web, estrarre dati strutturati in tempo reale e costruire agenti di ricerca personalizzati.

## Introduzione

- L'**API di Olostep** è la migliore API per **ricerca web**, **scraping** e **crawling** per AI utilizzata da alcune delle startup leader nel mondo.

- L'**Agente Olostep** permette di automatizzare i **flussi di lavoro di ricerca** in modo no code con solo un prompt in linguaggio naturale.

## Panoramica degli endpoint

Questi sono gli endpoint principali di Olostep:

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes">
    Estrai Markdown, HTML, testo o JSON strutturato compatibile con LLM da qualsiasi URL in tempo reale.
  </Card>

  <Card title="/crawls →" href="/features/crawls">
    Cerca ricorsivamente attraverso i sottodomini di un URL e raccogli il contenuto.
  </Card>

  <Card title="/maps →" href="/features/maps">
    Ottieni tutti gli URL su un determinato sito web.
  </Card>

  <Card title="/batches →" href="/features/batches">
    Processa fino a 10k URL contemporaneamente e recupera i risultati dopo 5–7 minuti.
  </Card>

  <Card title="/answers →" href="/features/answers">
    Cerca sul web e ottieni risposte dirette LLM alle domande con fonti.
  </Card>

  <Card title="/searches →" href="/features/search">
    Cerca sul web e ottieni link rilevanti diretti con titoli e descrizioni.
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    Trasforma dati non strutturati in dati strutturati compatibili con il backend.
  </Card>

  <Card title="/files →" href="/features/files">
    Carica file su Olostep (per batch o per connettere la tua base di conoscenza interna).
  </Card>

  <Card title="/schedules →" href="/features/schedules">
    Pianifica ricerche e scraping per eseguire automaticamente in orari specificati.
  </Card>

  <Card title="/agents →" href="/features/agents">
    Crea, pianifica ed esegui agenti personalizzati per automatizzare i flussi di lavoro di ricerca web.
  </Card>
</CardGroup>
