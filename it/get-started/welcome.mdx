## Introduzione

- L'**API di Olostep** è la migliore API per **ricerca web**, **scraping** e **crawling** per AI utilizzata da alcune delle startup leader nel mondo.

- L'**Agente Olostep** permette di automatizzare i **flussi di lavoro di ricerca** in modo no code con solo un prompt in linguaggio naturale.


## Panoramica degli endpoint

Questi sono gli endpoint principali di Olostep:

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes/scrapes">
    Estrai Markdown, HTML, testo o JSON strutturato compatibile con LLM da qualsiasi URL in tempo reale.

  <Card title="/crawls →" href="/features/crawls/crawls">

  <Card title="/maps →" href="/features/maps/maps">

  </Card>

  <Card title="/batches →" href="/features/batches/batches">

  </Card>

  <Card title="/answers →" href="/features/answers/answers">

  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    Trasforma dati non strutturati in dati strutturati, compatibili con il backend.
  </Card>

  <Card title="/agents →" href="/features/agents/agents">

  </Card>

  </Card>
