Per impostazione predefinita, Olostep supporta la restituzione di contenuti in vari formati, tra cui:
- HTML
- Testo
- PDF grezzo
- Markdown

**Per alcuni casi d'uso potresti non aver bisogno dell'intero contenuto ma solo di dati specifici, strutturati e puliti in formato JSON.**

Per ottenere questo, puoi utilizzare:

- **[Parser](./parsers)**: Ideali per lo scraping di siti web ad alto volume, coerente e ricorrente
- **[Estrazione LLM](./llm-extraction)**: Ideale per esigenze di estrazione flessibili o siti web con strutture in cambiamento.

Entrambi i metodi forniscono dati JSON puliti e strutturati che possono essere utilizzati immediatamente nelle tue applicazioni senza ulteriori elaborazioni.
