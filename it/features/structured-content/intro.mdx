Per impostazione predefinita, Olostep supporta la restituzione dei contenuti in vari formati tra cui:
- HTML
- Testo
- PDF grezzo
- Markdown

**Per alcuni casi d'uso potresti non aver bisogno dell'intero contenuto ma solo di dati specifici, strutturati e puliti in formato JSON.**

Per ottenere questo, puoi utilizzare:

- **[Parser](./parsers)**: Ideali per scraping di siti web ad alto volume, coerenti e ricorrenti
- **[Estrazione LLM](./llm-extraction)**: Ideale per esigenze di estrazione flessibili o siti web con strutture in cambiamento.

Per ottenere l'accesso alla funzione di Estrazione LLM, contattaci a info@olostep.com

Entrambi i metodi forniscono dati JSON puliti e strutturati che possono essere immediatamente utilizzati nelle tue applicazioni senza ulteriori elaborazioni.
