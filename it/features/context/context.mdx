Con i contesti puoi passare cookie personalizzati all'API. Questo ti consente di riutilizzare cookie, autenticazione e dati memorizzati per alimentare le automazioni AI.

### Creare un Contesto

Ecco come puoi creare un contesto per passare cookie personalizzati all'API:

1. Installa il plugin del browser Olostep da qui: [Olostep browser plugin](https://chromewebstore.google.com/detail/olostep/mjalbmdidmklbojfgakdpbifkgjjilkh)
2. Visita il sito web per il quale vuoi salvare i cookie/autenticazione
3. Accedi al sito web con le credenziali che vuoi salvare
4. Clicca sull'icona del plugin Olostep e seleziona "Invia Contesto"
5. Copia il context_id (che sarà qualcosa come `ctx_yylqbfayto`) dal plugin e usalo nelle tue richieste API

### Utilizzare il Contesto nelle richieste API

```python
import requests
import json

url = "https://api.olostep.com/v1/scrapes"

payload = {
    "url_to_scrape": "https://www.feizy.com/account-dashboard",
    "formats": ["html", "markdown"],
    "remove_css_selectors": "none",
    "country": "US",
    "context": {
        "id": "ctx_yylqbfayto"
    }
}

headers = {
    "Authorization": "Bearer <API_KEY>",
    "Content-Type": "application/json"
}

response = requests.request("POST", url, json=payload, headers=headers)

# Analizza la risposta JSON
data = response.json()

# Stampa il JSON con la corretta indentazione
print(json.dumps(data, indent=4))
```
