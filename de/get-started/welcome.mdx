Olostep bietet KI eine Möglichkeit, das Web zu durchsuchen, strukturierte Daten in Echtzeit zu extrahieren und benutzerdefinierte Forschungsagenten zu erstellen.

## Einführung

- Die **Olostep API** ist die beste **Websuch-, Scraping-** und **Crawling-API** für KI, die von einigen der führenden Startups der Welt genutzt wird.

- Der **Olostep Agent** ermöglicht die Automatisierung von **Forschungsabläufen** auf eine No-Code-Weise mit nur einem natürlichen Sprachbefehl.

## Überblick über die Endpunkte

Dies sind die Kernendpunkte von Olostep:

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes">
    Extrahiere LLM-freundliches Markdown, HTML, Text oder strukturiertes JSON von jeder URL in Echtzeit.
  </Card>

  <Card title="/crawls →" href="/features/crawls">
    Durchsuche rekursiv die Subdomains einer URL und sammle den Inhalt.
  </Card>

  <Card title="/maps →" href="/features/maps">
    Erhalte alle URLs auf einer bestimmten Website.
  </Card>

  <Card title="/batches →" href="/features/batches">
    Verarbeite bis zu 10.000 URLs gleichzeitig und erhalte die Ergebnisse nach 5–7 Minuten.
  </Card>

  <Card title="/answers →" href="/features/answers">
    Durchsuche das Web und erhalte direkte LLM-Antworten auf Fragen mit Quellen.
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    Verwandle unstrukturierte Daten in strukturierte, backend-kompatible Daten.
  </Card>

  <Card title="/agents →" href="/features/agents">
    Erstelle, plane und führe benutzerdefinierte Agenten aus, um Web-Forschungsabläufe zu automatisieren.
  </Card>

  <Card title="/files →">
    Lade Dateien zu Olostep hoch (für Batches oder um deine interne Wissensdatenbank zu verbinden).
  </Card>

  <Card title="/sandboxes →">
    Starte eine VM in unserer Cloud, damit dein Agent Computer-Modelle verwenden und mit dem Web und älterer Software interagieren kann. Schreibe uns an info@olostep.com für Zugang.
  </Card>
</CardGroup>
