- Mehrstufige Workflows mit geplanter Ausführung und Benachrichtigungen

## Einen Agenten erstellen

Erstellen Sie einen Agenten mit einem natürlichen Sprachprompt und einem Zielmodell.

<CodeGroup>

```python Python
API_URL = 'https://api.olostep.com/v1/agents' # Endpoint für ausgewählte Kunden verfügbar
API_KEY = '<YOUR_API_KEY>'

headers = {
  'Authorization': f'Bearer {API_KEY}', 
  'Content-Type': 'application/json'
}

data = {
  "prompt": """
    Finden Sie Portfoliounternehmen von https://www.vcsheet.com/funds und 
    schreiben Sie in ein Google Sheet mit den Spalten (Fondsname, Fonds-Website-URL, 
    Fonds-LinkedIn-URL, Portfoliounternehmensname, Portfoliounternehmens-URL, 
    Portfoliounternehmens-LinkedIn-URL). Führen Sie dies wöchentlich am Montag um 9:00 Uhr 
    aus und senden Sie eine E-Mail an steve@example.com, wenn neue Unternehmen hinzugefügt werden.
  """,
  "model": "gpt-4.1"
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()
print(result)
```

```js Node
const res = await fetch('https://api.olostep.com/v1/agents', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer <YOUR_API_KEY>', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'Finden Sie Portfoliounternehmen von https://www.vcsheet.com/funds und schreiben Sie in ein Google Sheet...',
    model: 'gpt-4.1'
  })
})
console.log(await res.json())
```

```bash cURL
curl -s -X POST "https://api.olostep.com/v1/agents" \
  -H "Authorization: Bearer $OLOSTEP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Finden Sie Portfoliounternehmen von https://www.vcsheet.com/funds und schreiben Sie in ein Google Sheet...",
    "model": "gpt-4.1"
  }'
```

</CodeGroup>

## Fähigkeiten

Der Olostep-Agent kann:

- **Web-Recherche**: Automatisches Suchen und Extrahieren von Daten aus Websites
- **Datenorganisation**: Strukturierung von Informationen in Tabellen, Datenbanken oder anderen Formaten
- **Geplante Ausführung**: Aufgaben in einem wiederkehrenden Zeitplan ausführen (täglich, wöchentlich, zu einer festgelegten Zeit)
- **Mehrstufige Workflows**: Komplexe, mehrteilige Forschungsaufgaben autonom durchführen
- **Benachrichtigungen**: E-Mail-Benachrichtigungen senden, wenn neue Daten gefunden werden oder Aufgaben abgeschlossen sind
- **Benutzerdefinierte Ausgabe**: Daten als CSV, JSON, Google Sheet oder direkt in Ihrer Datenbank zurückgeben
