title: "Batch-Elemente"
description: "Ruft die Liste der für einen Batch verarbeiteten Elemente ab. Sie können dann die `retrieve_id` verwenden, um den Inhalt mit dem Retrieve-Endpunkt zu erhalten."
openapi: "de/openapi/batches.json GET /v1/batches/{batch_id}/items"
---

<Tip>
**Metadaten:** Wenn Sie `metadata` an einzelne Elemente angehängt haben, als Sie den [Batch erstellt haben](/api-reference/batches/create), wird es mit jedem Element in der Antwort zurückgegeben.
</Tip>
