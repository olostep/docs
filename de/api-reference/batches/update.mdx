---
title: "Batch aktualisieren"
description: "Aktualisiert die Metadaten für einen bestimmten Batch. Nach der Erstellung des Batches können nur die Metadaten aktualisiert werden."
openapi: "de/openapi/batches.json PATCH /v1/batches/{batch_id}"
---

<Tip>
**Merge-Semantik:** Metadatenaktualisierungen folgen dem Ansatz von Stripe – neue Schlüssel werden hinzugefügt, bestehende Schlüssel werden aktualisiert, und auf leeren String `""` gesetzte Schlüssel werden gelöscht.
</Tip>
