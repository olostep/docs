Olostepは、AIがウェブを検索し、リアルタイムで構造化されたデータを抽出し、カスタムの研究エージェントを構築する方法を提供します。

## はじめに

- **Olostep API**は、世界の主要なスタートアップが使用するAI向けの最高の**ウェブ検索**、**スクレイピング**、**クロール**APIです。

- **Olostep Agent**は、自然言語のプロンプトだけで**研究ワークフロー**をコードなしで自動化することができます。

## エンドポイントの概要

これらはOlostepの主要なエンドポイントです：

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes">
    任意のURLからリアルタイムでLLM対応のMarkdown、HTML、テキスト、または構造化されたJSONを抽出します。
  </Card>

  <Card title="/crawls →" href="/features/crawls">
    URLのサブドメインを再帰的に検索し、コンテンツを収集します。
  </Card>

  <Card title="/maps →" href="/features/maps">
    特定のウェブサイト上のすべてのURLを取得します。
  </Card>

  <Card title="/batches →" href="/features/batches">
    最大10kのURLを同時に処理し、5–7分後に結果を取得します。
  </Card>

  <Card title="/answers →" href="/features/answers">
    ウェブを検索し、ソース付きの質問に対する直接のLLM回答を取得します。
  </Card>

  <Card title="/searches →" href="/features/search">
    ウェブを検索し、タイトルと説明付きの直接関連リンクを取得します。
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    非構造化データを構造化され、バックエンド互換のデータに変換します。
  </Card>

  <Card title="/files →" href="/features/files">
    Olostepにファイルをアップロードします（バッチ用または内部ナレッジベースに接続するため）。
  </Card>

  <Card title="/schedules →" href="/features/schedules">
    指定された時間に自動的に実行されるように検索とスクレイプをスケジュールします。
  </Card>

  <Card title="/agents →" href="/features/agents">
    カスタムエージェントを作成、スケジュール、実行してウェブ研究ワークフローを自動化します。
  </Card>
</CardGroup>
