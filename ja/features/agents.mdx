Olostep エージェント API を使用すると、自律的なリサーチエージェントを作成し、データパイプラインや検索タスクをスケジュールに基づいて自動化し、構造化された結果を提供できます。

- サイトを跨いだウェブリサーチと自動抽出
- スケジュール実行と通知を伴うマルチステップワークフロー
- JSON/CSV/Sheets/DB への出力

利用可能性と詳細については、[info@olostep.com](mailto:info@olostep.com) または [Contact Sales](https://www.olostep.com/contact-sales) までお問い合わせください。

## エージェントの作成

自然言語プロンプトとターゲットモデルを使用してエージェントを作成します。

<CodeGroup>

```python Python
API_URL = 'https://api.olostep.com/v1/agents' # 選ばれた顧客が利用可能なエンドポイント
API_KEY = '<YOUR_API_KEY>'

headers = {
  'Authorization': f'Bearer {API_KEY}', 
  'Content-Type': 'application/json'
}

data = {
  "prompt": """
    https://www.vcsheet.com/funds からポートフォリオ企業を見つけて、
    Google シートに (Fund Name, Fund Website URL, Fund LinkedIn URL, 
    Portfolio Company Name, Portfolio Company URL, Portfolio Company LinkedIn URL) 
    の列を含めて書き込みます。毎週月曜日の午前9時に実行し、新しい企業が
    追加された場合は steve@example.com にメールを送信します。
  """,
  "model": "gpt-4.1"
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()
print(result)
```

```js Node
const res = await fetch('https://api.olostep.com/v1/agents', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer <YOUR_API_KEY>', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'https://www.vcsheet.com/funds からポートフォリオ企業を見つけて Google シートに書き込みます...',
    model: 'gpt-4.1'
  })
})
console.log(await res.json())
```

```bash cURL
curl -s -X POST "https://api.olostep.com/v1/agents" \
  -H "Authorization: Bearer $OLOSTEP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "https://www.vcsheet.com/funds からポートフォリオ企業を見つけて Google シートに書き込みます...",
    "model": "gpt-4.1"
  }'
```

</CodeGroup>

## 機能

Olostep エージェントは以下のことが可能です：

- **ウェブリサーチ**: 自動的にウェブサイトからデータを検索し抽出
- **データ整理**: 情報をスプレッドシート、データベース、その他の形式に構造化
- **スケジュール実行**: 定期的なスケジュールでタスクを実行（毎日、毎週、事前に定義された時間）
- **マルチステップワークフロー**: 複雑で多段階のリサーチタスクを自律的に実行
- **通知**: 新しいデータが見つかったりタスクが完了したときにメールアラートを送信
- **カスタム出力**: データを CSV、JSON、Google シート、または直接データベースに返す
