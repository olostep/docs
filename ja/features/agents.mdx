Olostep エージェント API を通じて、自動データパイプラインや検索タスクをスケジュールに基づいて自動化し、構造化された結果を提供する自律型リサーチエージェントを作成できます。

- サイトを跨いだウェブリサーチと自動抽出
- スケジュール実行と通知を伴うマルチステップワークフロー
- JSON/CSV/Sheets/DB への出力

利用可能性と詳細については、[info@olostep.com](mailto:info@olostep.com) までご連絡いただくか、[営業に問い合わせる](https://www.olostep.com/contact-sales)をご覧ください。

## エージェントを作成する

自然言語プロンプトとターゲットモデルを使用してエージェントを作成します。

<CodeGroup>

```python Python
API_URL = 'https://api.olostep.com/v1/agents' # 選択された顧客に提供されるエンドポイント
API_KEY = '<YOUR_API_KEY>'

headers = {
  'Authorization': f'Bearer {API_KEY}', 
  'Content-Type': 'application/json'
}

data = {
  "prompt": """
    https://www.vcsheet.com/funds からポートフォリオ企業を見つけて、Google シートに (Fund Name, Fund Website URL, 
    Fund LinkedIn URL, Portfolio Company Name, Portfolio Company URL, 
    Portfolio Company LinkedIn URL) の列で書き込みます。毎週月曜日の午前9時に実行し、新しい企業が追加されたときに steve@example.com にメールを送信します。
  """,
  "model": "gpt-4.1"
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()
print(result)
```

```js Node
const res = await fetch('https://api.olostep.com/v1/agents', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer <YOUR_API_KEY>', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'Find portfolio companies from https://www.vcsheet.com/funds and write to a Google Sheet...',
    model: 'gpt-4.1'
  })
})
console.log(await res.json())
```

```bash cURL
curl -s -X POST "https://api.olostep.com/v1/agents" \
  -H "Authorization: Bearer $OLOSTEP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Find portfolio companies from https://www.vcsheet.com/funds and write to a Google Sheet...",
    "model": "gpt-4.1"
  }'
```

</CodeGroup>

## 機能

Olostep エージェントは以下のことができます：

- **ウェブリサーチ**: ウェブサイトからデータを自動的に検索および抽出
- **データ整理**: 情報をスプレッドシート、データベース、その他の形式に構造化
- **スケジュール実行**: 定期的なスケジュールでタスクを実行（毎日、毎週、事前に定義された時間に）
- **マルチステップワークフロー**: 複雑で多段階のリサーチタスクを自律的に実行
- **通知**: 新しいデータが見つかったときやタスクが完了したときにメールアラートを送信
- **カスタム出力**: データを CSV、JSON、Google シート、または直接データベースに返す
