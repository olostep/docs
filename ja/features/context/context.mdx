コンテキストを使用すると、カスタムクッキーをAPIに渡すことができます。これにより、クッキー、認証、および保存されたデータを再利用してAI自動化を実行できます。

### コンテキストの作成

APIにカスタムクッキーを渡すためのコンテキストを作成する方法は次のとおりです：

1. こちらからOlostepブラウザプラグインをインストールします: [Olostepブラウザプラグイン](https://chromewebstore.google.com/detail/olostep/mjalbmdidmklbojfgakdpbifkgjjilkh)
2. クッキー/認証を保存したいウェブサイトを訪問します
3. 保存したい資格情報でウェブサイトにログインします
4. Olostepプラグインアイコンをクリックし、「Submit Context」を選択します
5. プラグインからcontext_id（例: `ctx_yylqbfayto`）をコピーし、APIリクエストで使用します

### APIリクエストでのコンテキストの使用

```python
import requests
import json

url = "https://api.olostep.com/v1/scrapes"

payload = {
    "url_to_scrape": "https://www.feizy.com/account-dashboard",
    "formats": ["html", "markdown"],
    "remove_css_selectors": "none",
    "country": "US",
    "context": {
        "id": "ctx_yylqbfayto"
    }
}

headers = {
    "Authorization": "Bearer <API_KEY>",
    "Content-Type": "application/json"
}

response = requests.request("POST", url, json=payload, headers=headers)

# JSONレスポンスを解析
data = response.json()

# 適切なインデントでJSONを表示
print(json.dumps(data, indent=4))
```
