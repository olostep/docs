デフォルトで、Olostepは以下のようなさまざまな形式でコンテンツを返すことをサポートしています：
- HTML
- テキスト
- 生のPDF
- Markdown

**いくつかのユースケースでは、コンテンツ全体ではなく、指定された構造化されたクリーンなデータをJSON形式で必要とするかもしれません。**

これを実現するためには、以下を使用できます：

- **[Parsers](./parsers)**: 大量で、一貫性があり、定期的なウェブサイトのスクレイピングに最適
- **[LLM Extraction](./llm-extraction)**: 柔軟な抽出ニーズや構造が変化するウェブサイトに最適

LLM Extraction機能へのアクセスを得るには、info@olostep.comまでご連絡ください。

どちらの方法も、追加の処理なしでアプリケーションで直ちに使用できるクリーンで構造化されたJSONデータを提供します。
