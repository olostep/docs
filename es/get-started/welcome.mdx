## Introducción

- La **API de Olostep** es la mejor API de **búsqueda web**, **scraping** y **crawling** para IA utilizada por algunas de las startups líderes en el mundo.

- El **Agente de Olostep** permite automatizar **flujos de trabajo de investigación** de manera sin código con solo un mensaje en lenguaje natural.


## Resumen de Endpoints

Estos son los endpoints principales de Olostep:

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes/scrapes">
    Extrae Markdown, HTML, texto o JSON estructurado compatible con LLM de cualquier URL en tiempo real.
  </Card>

  <Card title="/crawls →" href="/features/crawls/crawls">
    Busca recursivamente a través de los subdominios de una URL y recopila el contenido.
  </Card>

  <Card title="/maps →" href="/features/maps/maps">
    Obtén todas las URLs en un sitio web determinado.
  </Card>

  <Card title="/batches →" href="/features/batches/batches">
    Procesa hasta 10k URLs simultáneamente y recupera los resultados después de 5–7 minutos.
  </Card>

  <Card title="/answers →" href="/features/answers/answers">
    Busca en la web y obtén respuestas directas de LLM a preguntas con fuentes.
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    Convierte datos no estructurados en datos estructurados compatibles con el backend.
  </Card>

  <Card title="/agents →" href="/features/agents/agents">
    Crea, programa y ejecuta agentes personalizados para automatizar flujos de trabajo de investigación web.
  </Card>

  <Card title="/files →">
    Sube archivos a Olostep (para lotes o para conectar tu base de conocimiento interna).
  </Card>

  <Card title="/sandboxes →">
    Inicia una VM en nuestra nube para permitir que tu agente use modelos de uso computacional e interactúe con la web y software heredado. Escríbenos a info@olostep.com para obtener acceso.
  </Card>
</CardGroup>
