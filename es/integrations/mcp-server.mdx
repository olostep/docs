El Servidor MCP de Olostep ofrece herramientas de acceso web para tus aplicaciones de IA. Permite que tu IA:
1. Busque en la web para basar la IA en datos del mundo real (hechos, documentos, eventos)
2. Obtenga el contenido markdown de cualquier página web
3. Descubra y filtre URLs relevantes dentro de un sitio web.

## Instalación
Necesitas tener una clave API. Obtén la clave API desde https://www.olostep.com/dashboard/api-keys.

Puedes usar nuestra URL alojada remotamente o ejecutar el servidor localmente.

### URL alojada remotamente

```bash
https://mcp.olostep.com/mcp
```

### Ejecutando con npx

```bash
env OLOSTEP_API_KEY=your-api-key npx -y olostep-mcp
```

### Instalación manual

```bash
npm install -g olostep-mcp
```

### Ejecutando en Claude Desktop

Añade esto a tu claude_desktop_config.json:

```bash
{
  "mcpServers": {
    "mcp-server-olostep": {
      "command": "npx",
      "args": ["-y", "olostep-mcp"],
      "env": {
        "OLOSTEP_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}
```

```bash
npx -y @smithery/cli install @olostep/olostep-mcp-server --client claude
```

### Ejecutando en Claude Code

Ejecuta esto en tu terminal

```bash
claude mcp add --transport http secure-api https://mcp.olostep.com/mcp \
  --header "Authorization: Bearer YOUR_API_KEY"
```

Alternativa (local / stdio)
```bash
claude mcp add --transport stdio --env OLOSTEP_API_KEY=YOUR_API_KEY olostep \
  -- npx -y olostep-mcp
```

#### Configuración manual

Para configurar Olostep MCP en Cursor:

- Abre Configuración de Cursor (haz clic en el icono de configuración en la parte superior derecha del IDE)
- Haz clic en MCP
- Haz clic en "+ Añadir nuevo servidor MCP global"
- Ingresa lo siguiente en el archivo mcp.json:
```bash
{
  "mcpServers": {
    "olostep": {
      "url": "https://mcp.olostep.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
```

Alternativa (local / stdio)
```bash
{
  "mcpServers": {
    "olostep-web": {
      "command": "npx",
      "args": ["-y", "olostep-mcp"],
      "env": {
        "OLOSTEP_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}
```
- Recuerda reemplazar `YOUR_API_KEY` con tu clave API real que puedes obtener del [tablero de Olostep](https://www.olostep.com/dashboard)
- Ahora puedes cerrar la pestaña de configuración. Si es necesario, actualiza la lista de servidores MCP para ver las nuevas herramientas.

> Si estás usando Windows y tienes problemas, intenta `cmd /c "set OLOSTEP_API_KEY=your-api-key && npx -y olostep-mcp"`

La próxima vez que uses Cursor, usará automáticamente el Olostep MCP cuando sea necesario. También puedes solicitar explícitamente usar el Olostep MCP cuando necesites información actualizada de la web.

## Herramientas Disponibles
- **scrape_website**: Extrae contenido de una sola URL. Soporta múltiples formatos y renderizado de JavaScript.
- **search_web**: Busca en la web una consulta dada y devuelve resultados estructurados (no IA, basado en parser).
- **answers**: Busca en la web y devuelve respuestas potenciadas por IA en la estructura JSON que desees, con fuentes y citas.
- **batch_scrape_urls**: Extrae hasta 10k URLs al mismo tiempo. Perfecto para extracción de datos a gran escala.
- **get_batch_results**: Recupera el estado y contenido extraído para un trabajo por lotes. Si está completo, obtiene elementos y recupera contenido para cada URL.
- **create_crawl**: Descubre y extrae autónomamente sitios web completos siguiendo enlaces desde una URL de inicio.
- **create_map**: Obtén todas las URLs de un sitio web. Extrae URLs para descubrimiento y análisis del sitio.
- **get_webpage_content**: Recupera el contenido de una página web en formato markdown.
- **get_website_urls**: Busca y recupera URLs relevantes de un sitio web.
- **google_search**: Recupera datos estructurados de los resultados de búsqueda de Google.
