## Instalación

### 1. Encuentra Olostep en Relay.app

Busca "Olostep" en Relay.app al crear un nuevo flujo de trabajo:

1. Ve a [Olostep en Relay.app](https://www.relay.app/apps/olostep/integrations)
2. Haz clic en "Comenzar gratis" o "Crear Flujo de Trabajo"
3. Busca "Olostep" en el selector de aplicaciones
4. Selecciona la aplicación Olostep

Alternativamente, visita la [página de integración de Olostep](https://www.relay.app/apps/olostep/integrations) directamente para explorar las acciones disponibles y crear tu primer flujo de trabajo.

### 2. Conecta Tu Cuenta

La primera vez que uses Olostep en un flujo de trabajo, se te pedirá que conectes tu cuenta:

1. Haz clic en "Conectar Olostep"
2. Ingresa tu clave API de Olostep
3. Autoriza la conexión

Obtén tu clave API desde el [Panel de Olostep](https://olostep.com/dashboard).

## Acciones Disponibles

### Scrape URL

Extrae contenido de una sola URL. Soporta múltiples formatos y renderizado de JavaScript.

**Casos de Uso:**
- Monitorea páginas específicas para cambios
- Extrae información de productos de sitios de comercio electrónico
- Recolecta datos de artículos de noticias o publicaciones de blogs
- Extrae contenido para agregación de contenido

**Configuración:**

<ParamField path="URL" type="string" required>
  URL del sitio web para hacer scraping (debe incluir http:// o https://)
</ParamField>

<ParamField path="Format" type="dropdown" default="Markdown">
  Elige formato: Markdown, HTML, JSON, o Texto Plano
</ParamField>

<ParamField path="Country Code" type="string">
  Código de país para contenido específico de ubicación (ej., "US", "GB", "CA")
</ParamField>

<ParamField path="Wait Before Scraping" type="integer">
  Tiempo de espera en milisegundos para el renderizado de JavaScript (0-10000)
</ParamField>

<ParamField path="Parser" type="string">
  ID de parser opcional para extracción especializada (ej., "@olostep/amazon-product")
</ParamField>

**Campos de Salida:**
- ID de Scrape
- URL Scrapeada
- Contenido Markdown
- Contenido HTML
- Contenido JSON
- Contenido de Texto
- Estado
- Marca de Tiempo
- URL de Captura de Pantalla (si está disponible)
- Metadatos de la Página

## Ventajas de Relay.app

Relay.app ofrece varias ventajas sobre las plataformas de automatización tradicionales:

### Flujos de Trabajo con Participación Humana

Añade pasos de revisión manual directamente en tus flujos de trabajo:

```
1. Olostep hace scraping del sitio web
2. Paso de Revisión Humana ← Revisar y aprobar datos
3. Guardar en la base de datos solo si se aprueba
```

Perfecto para:
- Control de calidad antes del procesamiento de datos
- Juicio humano sobre contenido extraído
- Flujos de trabajo de aprobación para datos sensibles
- Revisar resúmenes generados por IA

### Funciones de IA Integradas

Usa IA directamente en tus flujos de trabajo sin integraciones externas:

- **Toma de Decisiones con IA**: Deja que la IA decida rutas de flujo de trabajo basadas en contenido extraído
- **Filtrado Inteligente**: Filtra y categoriza automáticamente los datos extraídos
- **Resumen de Contenidos**: Resume artículos largos antes de almacenarlos
- **Extracción de Datos**: Usa IA para extraer datos estructurados de contenido no estructurado

### Mejor UX y Pruebas

La interfaz amigable de Relay.app facilita la construcción de flujos de trabajo:

- Información clara de pruebas
- Mensajes de error útiles
- Constructor visual de flujos de trabajo
- Depuración y resolución de problemas fáciles

## Ejemplos Populares de Flujos de Trabajo

### Monitoreo de Precios de E-commerce con Revisión

Monitorea precios de la competencia con aprobación humana:

```
Disparador: Programar (Cada hora)
↓
Acción: Olostep - Scrape URL
  - URL: Página de producto de la competencia
  - Formato: JSON
  - Parser: @olostep/amazon-product
↓
Paso de Revisión Humana
  - Revisar cambio de precio
  - Aprobar o rechazar
↓
Acción: Slack - Enviar Mensaje (Si se aprueba)
  - Alerta: "El precio cambió a $\{\{price\}\}"
```

### Agregación de Contenido con IA

Agrupa y mejora contenido de múltiples fuentes:

```
Disparador: Google Sheets - Nueva Fila
↓
Acción: Olostep - Scrape URL
  - URL: \{\{URL de la hoja\}\}
  - Formato: Markdown
↓
Acción de IA: Resumir Contenido
  - Generar resumen
↓
Acción: Airtable - Crear Registro
  - Almacenar artículo con resumen de IA
```

### Pipeline de Enriquecimiento de Leads

Enriquece datos de leads con información web e IA:

```
Disparador: HubSpot - Nuevo Contacto
↓
Acción: Olostep - Scrape URL
  - URL: \{\{Sitio web de la empresa\}\}
  - Formato: Markdown
↓
Acción de IA: Extraer Información de la Empresa
  - Extraer: tamaño de la empresa, industria, productos
↓
Paso de Revisión Humana
  - Verificar precisión de la extracción de IA
↓
Acción: HubSpot - Actualizar Contacto
  - Añadir datos enriquecidos verificados
```

## Integración con Aplicaciones Populares

### Google Sheets

Perfecto para la recolección y seguimiento de datos:

```
1. Olostep hace scraping del sitio web
2. IA - Procesa y formatea datos
3. Revisión Humana (opcional)
4. Google Sheets - Crear/Actualizar Fila
```

**Casos de Uso:**
- Hojas de seguimiento de precios
- Bases de datos de enriquecimiento de leads
- Inventario de contenido
- Hojas de análisis de la competencia

### Airtable

Construye potentes bases de datos con datos extraídos:

```
1. Olostep hace scraping o rastreo
2. IA - Extrae datos estructurados
3. Airtable - Crear Registros
```

**Casos de Uso:**
- Catálogos de productos
- Bases de datos de investigación
- Calendarios de contenido
- Bases de datos de enlaces

### Slack

Recibe notificaciones instantáneas con flujos de trabajo de aprobación:

```
1. Olostep monitorea página
2. Filtro - Verifica cambios
3. Paso de Aprobación Humana
4. Slack - Enviar Mensaje (si se aprueba)
```

**Casos de Uso:**
- Alertas de caída de precios (con aprobación)
- Notificaciones de actualización de contenido
- Monitoreo de errores
- Resúmenes diarios

### HubSpot / Salesforce

Enriquece automáticamente datos de CRM:

```
1. Nuevo contacto añadido
2. Olostep hace scraping del sitio web de la empresa
3. IA extrae información clave
4. Paso de Revisión Humana
5. CRM - Actualizar contacto (si se aprueba)
```

**Casos de Uso:**
- Enriquecimiento de leads con control de calidad
- Investigación de empresas
- Inteligencia competitiva
- Mapeo de cuentas

### Notion

Construye bases de conocimiento con contenido revisado:

```
1. Olostep rastrea documentación
2. IA - Analiza y estructura contenido
3. Paso de Revisión Humana
4. Notion - Crear Páginas (contenido aprobado)
```

**Casos de Uso:**
- Espejos de documentación
- Repositorios de investigación
- Bibliotecas de contenido
- Wikis de equipo

## Solución de Problemas

<AccordionGroup>
  <Accordion title="Autenticación Fallida">
    **Error**: "Clave API inválida"
    
    **Soluciones**:
    - Verifica la clave API desde el [panel](https://olostep.com/dashboard)
    - Asegúrate de que no haya espacios adicionales en la clave API
    - Reconecta tu cuenta de Olostep en Relay.app
    - Verifica que la clave API esté activa
  </Accordion>

  <Accordion title="Scrape Devuelve Contenido Vacío">
    **Error**: Los campos de contenido están vacíos
    
    **Soluciones**:
    - Aumenta el tiempo de "Wait Before Scraping"
    - Verifica si el sitio web requiere inicio de sesión
    - Prueba con un formato diferente (HTML vs Markdown)
    - Verifica que la URL sea accesible
    - Verifica si el sitio bloquea el acceso automatizado
  </Accordion>

  <Accordion title="Límite de Tasa Excedido">
    **Error**: "Límite de tasa excedido"
    
    **Soluciones**:
    - Espacia las ejecuciones de flujo de trabajo con retrasos
    - Usa procesamiento por lotes en lugar de scrapes individuales
    - Mejora tu plan de Olostep
    - Verifica el límite de tasa en el panel
  </Accordion>

  <Accordion title="URL No Scrapeada">
    **Error**: URLs específicas fallan al hacer scraping
    
    **Soluciones**:
    - Verifica el formato de la URL (incluye http:// o https://)
    - Verifica si la URL requiere autenticación
    - Prueba la URL en el navegador primero
    - Prueba con el parámetro de país
    - Contacta al soporte para dominios bloqueados
  </Accordion>
</AccordionGroup>

## Relay.app vs Zapier

Relay.app es una alternativa moderna a Zapier con varias ventajas:

**Ventajas de Relay.app:**
- **Participación Humana**: Pasos de revisión manual integrados
- **IA Integrada**: Funciones de IA integradas en los flujos de trabajo
- **Mejor UX**: Interfaz más intuitiva y mensajes de error más claros
- **Mejor Precio**: Más rentable para muchos casos de uso
- **Integraciones Más Profundas**: Integraciones más completas con aplicaciones populares

**Cuándo Elegir Zapier:**
- Necesitas integración con aplicaciones menos comunes
- Prefieres una plataforma más establecida
- Requieres características específicas solo de Zapier

Para comparaciones detalladas, ve la [comparación de Zapier de Relay.app](https://www.relay.app/blog/zapier-alternatives).

## Precios

Olostep cobra según el uso de la API, independiente de Relay.app:

- **Scrapes**: Pago por scrape
- **Lotes**: Pago por URL en lote
- **Rastreos**: Pago por página rastreada
- **Mapas**: Pago por operación de mapa

Consulta los precios actuales en [olostep.com/pricing](https://olostep.com/pricing).

**Plan de Relay.app**: También necesitas un plan activo de Relay.app para ejecutar flujos de trabajo.
