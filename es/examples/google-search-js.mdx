# Scraper de Búsqueda de Google con Olostep

Esta guía demuestra cómo usar la API de Olostep para extraer resultados de búsqueda de Google y convertirlos en datos JSON estructurados. Esto es particularmente útil para automatizar tareas de investigación, recopilar inteligencia competitiva o construir aplicaciones que requieran datos de búsqueda.

## Cómo Funciona

El ejemplo a continuación en Javascript muestra cómo buscar la URL de un perfil de LinkedIn de una persona específica (Patrick Collison) usando la búsqueda de Google y el parser de búsqueda de Google de Olostep `@olostep/google-search`

```javascript
async function scrapeGoogleSearch(apiKey, query = "site%3Alinkedin.com+Patrick+Collison") {
  const endpoint = "https://api.olostep.com/v1/scrapes";

  const payload = {
    "formats": ["json"],
    "parser": {"id": "@olostep/google-search"},
    "url_to_scrape": `https://www.google.com/search?q=${encodeURIComponent(query)}&gl=us&hl=en`,
    "wait_before_scraping": 0,
  };

  const headers = {
    "Authorization": `Bearer ${apiKey}`,
    "Content-Type": "application/json"
  };

  try {
    const response = await fetch(endpoint, {
      method: "POST",
      headers: headers,
      body: JSON.stringify(payload)
    });

    const data = await response.json();
    console.log(JSON.stringify(data, null, 4));
    return data;
  } catch (error) {
    console.error("Error:", error);
    throw error;
  }
}

// Reemplaza <API_KEY> con tu clave API de Olostep real
scrapeGoogleSearch("<API_KEY>");
```

La respuesta contiene:

- **Información básica de la solicitud**: `id`, `object`, marca de tiempo `created`, `url_to_scrape`
- **Objeto de resultado** con URLs para acceder a diferentes formatos de los datos
- **json_content** con resultados de búsqueda estructurados incluyendo:
- `searchParameters`: Información sobre la consulta de búsqueda
- `knowledgeGraph`: Información detallada sobre el sujeto de búsqueda (cuando esté disponible)
- `organic`: Lista de resultados de búsqueda con título, enlace, posición y fragmento
- `peopleAlsoAsk`: Preguntas relacionadas que los usuarios suelen buscar
- `relatedSearches`: Consultas de búsqueda relacionadas sugeridas

`json_content` es la parte principal de la respuesta con los resultados de búsqueda estructurados. Puedes acceder al contenido JSON directamente desde la respuesta o usar la URL alojada proporcionada en la respuesta.

## Ejemplos de Uso

### 1. Encontrar Perfiles de LinkedIn

En el ejemplo anterior, estamos buscando el perfil de LinkedIn de Patrick Collison usando la consulta de búsqueda `site:linkedin.com Patrick Collison`. Esta técnica se puede usar para encontrar perfiles profesionales de cualquier persona.

### 2. Investigar Empresas

Puedes modificar la consulta para buscar información de empresas:

```javascript
// Buscar información de la empresa
scrapeGoogleSearch(apiKey, "Stripe company information revenue");
```

### 3. Rastrear Artículos de Noticias

Encuentra noticias recientes sobre un tema específico:

```javascript
// Buscar noticias recientes sobre IA
scrapeGoogleSearch(apiKey, "artificial intelligence news after:2023-01-01");
```

### 4. Análisis Competitivo

Monitorea la presencia en línea de los competidores:

```javascript
// Buscar menciones de competidores
scrapeGoogleSearch(apiKey, "\"Company X\" vs \"Company Y\" comparison");
```

## Notas Importantes

4. **Parámetros de Búsqueda**: Los parámetros `gl=us` y `hl=en` establecen la geolocalización en EE.UU. y el idioma en inglés. Ajusta estos según sea necesario.

## Conclusión

Una vez que tengas los datos de los resultados de búsqueda, puedes:

1. Analizar puntos de datos específicos de interés
2. Almacenar los resultados en una base de datos
3. Construir una interfaz de búsqueda personalizada
4. Desencadenar acciones basadas en los hallazgos de búsqueda
5. Integrar con otras APIs o servicios
