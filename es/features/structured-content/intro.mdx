Por defecto, Olostep admite devolver contenido en varios formatos, incluyendo:
- HTML
- Texto
- PDF sin procesar
- Markdown

**Para algunos casos de uso, es posible que no necesites todo el contenido, sino solo datos especificados, estructurados y limpios en formato JSON.**

Para lograr esto, puedes usar:

- **[Parsers](./parsers)**: Ideal para scraping de sitios web de alto volumen, consistente y recurrente
- **[Extracción LLM](./llm-extraction)**: Ideal para necesidades de extracción flexibles o sitios web con estructuras cambiantes.

Ambos métodos proporcionan datos JSON limpios y estructurados que pueden ser utilizados inmediatamente en tus aplicaciones sin procesamiento adicional.
