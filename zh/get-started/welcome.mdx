## 介绍

- **Olostep API** 是一些全球领先初创公司使用的最佳 **网络搜索**、**抓取** 和 **爬虫** API。

- **Olostep Agent** 允许通过自然语言提示以无代码方式自动化 **研究工作流程**。


## 端点概览

以下是 Olostep 的核心端点：

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes">
    从任何 URL 实时提取适合 LLM 的 Markdown、HTML、文本或结构化 JSON。
  </Card>

  <Card title="/crawls →" href="/features/crawls">
    递归搜索 URL 的子域并收集内容。
  </Card>

  <Card title="/maps →" href="/features/maps">
    获取某个网站上的所有 URL。
  </Card>

  <Card title="/batches →" href="/features/batches">
    同时处理多达 10k 个 URL，并在 5-7 分钟后检索结果。
  </Card>

  <Card title="/answers →" href="/features/answers">
    搜索网络并获取带有来源的直接 LLM 答案。
  </Card>

  <Card title="/searches →" href="/features/search">
    搜索网络并获取带有标题和描述的直接相关链接。
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    将非结构化数据转换为结构化、后端兼容的数据。
  </Card>

  <Card title="/files →" href="/features/files">
    将文件上传到 Olostep（用于批处理或连接你的内部知识库）。
  </Card>

  <Card title="/schedules →" href="/features/schedules">
    安排搜索和抓取在指定时间自动运行。
  </Card>

  <Card title="/agents →" href="/features/agents">
    创建、安排和运行自定义代理以自动化网络研究工作流程。
  </Card>
</CardGroup>
