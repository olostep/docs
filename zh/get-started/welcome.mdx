## 介绍

- **Olostep API** 是全球领先初创公司使用的最佳 **网络搜索**、**抓取** 和 **爬虫** API。

- **Olostep Agent** 允许以无代码方式通过自然语言提示自动化 **研究工作流程**。

## 端点概览

以下是 Olostep 的核心端点：

<CardGroup cols={1}>

  <Card title="/scrapes →" href="/features/scrapes/scrapes">
    从任何 URL 实时提取适合 LLM 的 Markdown、HTML、文本或结构化 JSON。
  </Card>

  <Card title="/crawls →" href="/features/crawls/crawls">
    递归搜索 URL 的子域并收集内容。
  </Card>

  <Card title="/maps →" href="/features/maps/maps">
    获取某个网站上的所有 URL。
  </Card>

  <Card title="/batches →" href="/features/batches/batches">
    并发处理多达 10k 个 URL，并在 5-7 分钟后检索结果。
  </Card>

  <Card title="/answers →" href="/features/answers/answers">
    搜索网络并获取带有来源的直接 LLM 答案。
  </Card>

  <Card title="/parsers →" href="/features/structured-content/parsers">
    将非结构化数据转换为结构化、后端兼容的数据。
  </Card>

  <Card title="/agents →" href="/features/agents/agents">
    创建、调度和运行自定义代理以自动化网络研究工作流程。
  </Card>

  <Card title="/files →">
    上传文件到 Olostep（用于批处理或连接您的内部知识库）。
  </Card>

  <Card title="/sandboxes →">
    在我们的云中启动一个 VM，让您的代理使用计算机模型并与网络和旧软件交互。请通过 info@olostep.com 联系我们以获取访问权限。
  </Card>
</CardGroup>
