通过上下文，你可以将自定义cookie传递给API。这使你能够重用cookie、身份验证和存储的数据来支持AI自动化。

### 创建上下文

以下是如何创建一个上下文以将自定义cookie传递给API：

1. 从这里安装Olostep浏览器插件：[Olostep浏览器插件](https://chromewebstore.google.com/detail/olostep/mjalbmdidmklbojfgakdpbifkgjjilkh)
2. 访问你想要保存cookie/身份验证的网站
3. 使用你想要保存的凭据登录该网站
4. 点击Olostep插件图标并选择“提交上下文”
5. 从插件中复制context_id（类似于`ctx_yylqbfayto`）并在你的API请求中使用

### 在API请求中使用上下文

```python
import requests
import json

url = "https://api.olostep.com/v1/scrapes"

payload = {
    "url_to_scrape": "https://www.feizy.com/account-dashboard",
    "formats": ["html", "markdown"],
    "remove_css_selectors": "none",
    "country": "US",
    "context": {
        "id": "ctx_yylqbfayto"
    }
}

headers = {
    "Authorization": "Bearer <API_KEY>",
    "Content-Type": "application/json"
}

response = requests.request("POST", url, json=payload, headers=headers)

# 解析JSON响应
data = response.json()

# 以适当的缩进打印JSON
print(json.dumps(data, indent=4))
```
