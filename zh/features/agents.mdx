## 创建代理

使用自然语言提示和目标模型创建代理。

<CodeGroup>

```python Python
API_URL = 'https://api.olostep.com/v1/agents' # 仅限选定客户使用的端点
API_KEY = '<YOUR_API_KEY>'

headers = {
  'Authorization': f'Bearer {API_KEY}', 
  'Content-Type': 'application/json'
}

data = {
  "prompt": """
    从 https://www.vcsheet.com/funds 找到投资组合公司，并写入 Google 表格，列包括（基金名称、基金网站 URL、基金 LinkedIn URL、投资组合公司名称、投资组合公司 URL、投资组合公司 LinkedIn URL）。每周一上午 9:00 运行，当有新公司添加时，发送邮件到 steve@example.com。
  """,
  "model": "gpt-4.1"
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()
print(result)
```

```js Node
const res = await fetch('https://api.olostep.com/v1/agents', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer <YOUR_API_KEY>', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: '从 https://www.vcsheet.com/funds 找到投资组合公司，并写入 Google 表格...',
    model: 'gpt-4.1'
  })
})
console.log(await res.json())
```

```bash cURL
curl -s -X POST "https://api.olostep.com/v1/agents" \
  -H "Authorization: Bearer $OLOSTEP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "从 https://www.vcsheet.com/funds 找到投资组合公司，并写入 Google 表格...",
    "model": "gpt-4.1"
  }'
```

</CodeGroup>

## 功能

Olostep 代理可以：

- **网络研究**：自动搜索并从网站提取数据
- **数据组织**：将信息结构化为电子表格、数据库或其他格式
- **计划执行**：按周期性计划运行任务（每日、每周、在预定时间）
- **多步骤工作流**：自主执行复杂的多部分研究任务
- **通知**：在发现新数据或任务完成时发送电子邮件提醒
- **自定义输出**：以 CSV、JSON、Google 表格或直接在你的数据库中返回数据
