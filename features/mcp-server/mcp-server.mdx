Olostep MCP Server offers best-in-class web access tooling to your AI apps with minimal setup. It allows your AI clients to:
1. Search web using google
2. Discover relevant URLs within a website.
3. Get web pages content in markdown

Setting Olostep MCP server is as simple as:

```bash
env OLOSTEP_API_KEY=your-api-key npx -y olostep-mcp
```

You can find complete details about Olostep MCP [here](https://github.com/olostep/olostep-mcp-server).
